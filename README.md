# FM-MOD

Cookie Clicker（ブラウザ版）向けの機能拡張・支援のFMsargeが開発したModメニューです。
ブラウザの開発者ツール（F12/コンソール）が制限されている環境（学校・企業等のPC）においても、ブックマークレット形式（お気に入りへの登録）により、1クリックで独立したUIメニューを起動することが可能です。

---

## 主要機能

* **WRINKLER ESP**
  画面上に存在する Wrinkler（虫）の内部データを走査。各個体が吸収したクッキー量および経過時間をリアルタイムで算出し、赤い追跡ラインと緑のテキストインジケータによって画面上に描画・可視化します。

* **GOD'S EYE**
  次に出現するゴールデンクッキーのタイマー（秒数）および、発生するバフ（効果）のログを事前に検知・表示します。

* **Infinity goldencookie**
  ゴールデンクッキーのクリック判定を擬似的に高速自動実行（毎秒約30回）し、各バフ効果を無限に重複・維持させます。
* **その他もろもろ　約20個の機能があります**

## 導入方法

1. **ソースコードの取得**
   本リポジトリ内の `FM-MOD_verX.X.X.txt` を開き、記述されているJavaScriptコードをすべてクリップボードにコピーします。

2. **ブックマークレットの登録**
   任意のブラウザで新しいブックマークを作成し、URL（アドレス）欄にコピーしたコードをそのまま貼り付けて保存します。

3. **実行**
   Cookie Clickerのゲームページを開いた状態で、登録したブックマークをクリックすると、画面上にFM-MODの制御メニューが展開されます。

## Overview (English)
FM-MOD is a powerful extension and cheat menu for the browser game "Cookie Clicker," developed by FMsarge. 
It features a standalone UI that bypasses environment restrictions (like blocked F12 developer tools) using a single-click bookmarklet.
Key features include Wrinkler ESP, Golden Cookie buff predictions (God's Eye), and automated infinite cookie clicking.
